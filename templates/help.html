Explanation:


With this application it is possible to simulate a titration of any mixture of acids and bases. The concentrations can be entered comma separated as it is shown with the default values. To enter pKs values, you have to write them comma separated between brackets [...]. A species can have as many pKs values as needed. If you have only bases or only acids, enter for the opposit "0" as a concentration and [0] as pKs value. pKs(Tit) is the pKs value of the titration agent. So if you use Sodiumhydroxide to titrate something just add [14]. To execute the simulation enter the compute button.
